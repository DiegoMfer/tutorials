## Available Tutorials

- **[RDF 1.2 Tutorial](rdf12_tutorial/)** - Learn about RDF 1.2 features and how to work with RDF data using rudof

## About

This repository contains practical tutorials for learning and using [rudof](https://github.com/rudof-project/rudof), an RDF data shapes library implemented in Rust.

## Contributing

We welcome contributions! If you'd like to add a new tutorial or improve existing ones:

1. Fork the repository
2. Create a new branch for your tutorial
3. Add your tutorial in a new directory
4. Update this README with a link to your tutorial
5. Submit a pull request
